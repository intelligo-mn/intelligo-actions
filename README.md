
# Intelligo Actions

Github action for Intelligo Framework. :rocket:

## Code in Master

Install the dependencies  
```bash
$ npm install
```

Build the typescript
```bash
$ npm run build
```

Run the tests :heavy_check_mark:  
```bash
$ npm test

```

## Usage:

```yaml
uses: intelligo-mn/intelligo-actions@v1
with:
  packages: '["my-package"]'
```
